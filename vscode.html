VSCode Extension

Full language support for ACDL files with syntax highlighting, live diagnostics, and preview

Features

Syntax Highlighting

Color-coded keywords, roles, templates, context variables, strings, and comments

Live Diagnostics

Parse errors shown as red underlines in real time as you type

Live Preview

Renders the visualization alongside the editor with automatic updates

Go-to-Definition

Click on prompt.X references to jump to label definitions

Bracket Matching

Auto-closing pairs and matching bracket highlighting

Code Folding

Collapse and expand { } blocks for easier navigation

Installation

Option 1: Command Line

# Install from the VSIX file
code --install-extension acdl-language-0.1.0.vsix

Option 2: VSCode UI

Step 1

Open VSCode and press Ctrl+Shift+P (or Cmd+Shift+P on Mac)

Step 2

Type Extensions: Install from VSIX... and select it

Step 3

Navigate to and select the acdl-language-0.1.0.vsix file

Step 4

Reload VSCode: Ctrl+Shift+P → Developer: Reload Window

Download VSIX

Usage

Once installed, the extension automatically activates for any file with the .acdl extension.

Preview Panel

To open the live preview panel:

  • Press Ctrl+Shift+P
  • Type ACDL: Show Preview
  • The visualization will appear in a side panel and update as you edit

Comment Toggling

Use Ctrl+/ (or Cmd+/ on Mac) to toggle line comments.
